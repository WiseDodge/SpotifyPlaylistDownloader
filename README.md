A lightweight, portable wrapper for [spotDL](https://github.com/spotDL/spotify-downloader) to download Spotify playlists seamlessly.

**Created by [Mr. Wise](https://github.com/WiseDodge)**

## Features
* **Zero Installation:** No need to install Python or FFmpeg. Just extract and run.
* **Smart Sync:** Skips songs you've already downloaded.
* **Auto-Repair:** Includes a self-healing launcher that backs itself up.

## Usage
1. Download the release `.zip` from the [Releases Page](https://github.com/WiseDodge/SpotifyPlaylistDownloader/releases).
2. Extract the folder to your Desktop (or anywhere you like).
3. Double-click `SpotifyDownloader.bat`.
4. Paste your Spotify Link (Playlists or Individual Song) and click **Download**.
